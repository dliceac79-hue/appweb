# LogiOps PHP Application

This repository contains the LogiOps PHP 8 application with MySQL (PDO) backend. Use the following pointers to verify the files are present:

- **Database schema:** `database.sql` at the project root.
- **Core app files:** under `app/` (`config.php`, `db.php`, `auth.php`, `helpers.php`, `credit.php`, `provider_credit.php`).
- **Public entry point and router:** `public/index.php`.
- **Pages:** `public/pages/` (login, dashboard, invoices, vendor bills, statements, etc.).
- **AJAX endpoints:** `public/ajax/` (invoices and vendor bill handlers).

Clone or download this repo and you should see the paths above in your working copy. If you do not see them, ensure you are on the latest commit (`Add initial LogiOps PHP application`).
